# 70mai Dash Cam Configurator

An app to modify the settings of the 70mai dash cam.

## Getting started

Building the app requires the Xamarin.Android extension to Visual Studio. You should always clean the solution before attempting to run the app. You may experience strange behaviour and crashes if you don't clean the solution first.

## Front page features

- ApkAuthorize
- SoundIndicator (get/set)
- Custom requests
